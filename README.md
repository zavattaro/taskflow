# TaskFlow

Projeto full-stack de gestão de tarefas para estudo prático com Angular, .NET e PostgreSQL.

## Objetivo
Construir uma aplicação onde usuários autenticados possam criar projetos, cadastrar tarefas e acompanhar o status de execução.

## Stack inicial
- Angular
- ASP.NET Core Web API
- .NET 10
- Entity Framework Core
- PostgreSQL
- Docker

## Estrutura do repositório
- `frontend/` aplicação Angular
- `backend/` API .NET
- `docs/` documentação do projeto

## Escopo inicial
O escopo funcional do MVP está documentado em `docs/mvp.md`.
